## Using this as a GitHub Pages site

1. **Create a new repo on GitHub** (e.g. `tech-setup-part2` or `yourname-quarto-portfolio`). Choose Public and optionally "Add a README".
2. **In a terminal, open this folder** (`challengeHandouts/TechSetupChallengePart2`), then run:
   ```bash
   git init
   git add .
   git commit -m "Initial commit - GitHub Pages site"
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git branch -M main
   git push -u origin main
   ```
3. **Turn on GitHub Pages:** Repo → **Settings** → **Pages** → **Source**: "Deploy from a branch" → Branch: **main** → Folder: **/ (root)** → Save.
4. After a minute or two, your site will be at `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/` (GitHub uses `index.html` from the root automatically).
